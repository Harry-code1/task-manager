# Task Manager (Python)

A simple command-line task manager written in Python.

## Features

* Add tasks
* Mark tasks as complete
* Delete tasks
* Optional due dates
* Tasks are saved using a JSON file

## How to run

1. Make sure Python 3 is installed
2. Clone the repository
3. Run:

```
python task_manager.py
```

## Data storage

Tasks are stored locally in a `tasks.json` file so they persist between runs.

## Purpose

This project was built to practise Python fundamentals, file handling, and JSON data storage.
